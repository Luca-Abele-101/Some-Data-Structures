## Repository contains subprojects:
- **Binary tree** data structure  
- **Stack** data structure  
- **Queue** data structure  

## Build

For building projects, use:
```
mkdir build && cd build
cmake ../
make
```

## Test

For testing projects after build, use:
```
ctest
```
